Diseñar y desarrollar una aplicación web con React en la que el usuario pueda registrar "tareas", sus prioridades y categorías.
Las tareas deben mostrarse ordenadas por su prioridad y deben poder filtrarse por categoría.
Las tareas deben poder cambiar de estado entre "pendiente" (estado inicial), "en proceso" y "finalizado". También se deben poder eliminar.
El diseño debe contemplar el maquetado de la interfaz de usuario, y los componentes con las props heredadas entre sí.
