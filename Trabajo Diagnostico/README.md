TP DIAGNOSTICO

Crear una aplicacion web donde el usuario pueda agregar peliculas, listarlas, mostrar sus detalles, ordenarlas y filtrarlas por categorias.

Cada pelicula debe contener un titulo, una categroria (seleccionable desde una lista predeterminada), una url a una imagen de portada, una descripcion y una valoracion.

Al mostrar el listado de peliculas se debe hacer de forma resumida, mostrando solo titulos y portadas. Al clickear se deberan mostrar los demas datos. El listado se debe poder filtrar por categoria y ordenar por valoración.
